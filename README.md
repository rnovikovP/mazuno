# mazuno
Arduino uno maze game

# using 
- U8G2 library for SH1107 I2C display 128X128
- 2 position xy joystic
- buzzer

# wires

display |  | UNO   
--- | --- | --- 
SDA | <--> | A4   
SCL | <--> | A5
GND | <--> | GND
VCC | <--> | VCC



Joystic | | UNO   
--- | --- | --- 
VRX | <--> | A1     
VRY | <--> | A0
GND | <--> | GND
VCC | <--> | VCC


Buzzer | | UNO  
--- | --- | --- 
GND | <--> | GND
VCC | <--> | D2
